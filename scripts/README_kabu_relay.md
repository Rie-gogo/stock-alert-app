# kabu STATION® API 板情報中継スクリプト セットアップガイド

## 概要

このスクリプトは、Windows上のkabuステーション®から板情報（オーダーブック）をリアルタイムで取得し、クラウドのWebアプリに転送します。

```
kabuステーション® → このスクリプト（Windows） → クラウドWebアプリ → ブラウザ画面
```

---

## 必要なもの

- Windows PC（kabuステーション®がインストール済み）
- Python 3.8以上
- kabuステーション® Professionalプラン（API利用可能な状態）

---

## セットアップ手順

### 1. Pythonのインストール確認

コマンドプロンプトを開いて以下を実行：

```
python --version
```

`Python 3.x.x` と表示されればOKです。表示されない場合は [python.org](https://www.python.org/downloads/) からインストールしてください。

### 2. 必要なライブラリをインストール

```
pip install requests websocket-client
```

### 3. スクリプトのダウンロード

`kabu_board_relay.py` をWindowsの任意のフォルダ（例: `C:\kabu_relay\`）に保存します。

### 4. APIパスワードを設定

`kabu_board_relay.py` をメモ帳で開き、以下の行を編集します：

```python
KABU_API_PASSWORD = "YOUR_API_PASSWORD_HERE"  # ← ここを変更
```

kabuステーション®のAPIシステム設定で設定したパスワードを入力してください。

### 5. 環境の設定（本番 or 検証）

```python
KABU_API_PORT = 18081  # 検証環境
# KABU_API_PORT = 18080  # 本番環境に切り替える場合はこちら
```

### 6. スクリプトを起動

kabuステーション®を起動した状態で、コマンドプロンプトを開いて実行：

```
cd C:\kabu_relay
python kabu_board_relay.py
```

以下のようなログが表示されれば正常動作しています：

## 自動起動の設定（オプション）

Windowsのタスクスケジューラを使って、PC起動時に自動でスクリプトを起動できます。

1. スタートメニューで「タスクスケジューラ」を検索して開く
2. 「タスクの作成」をクリック
3. 「トリガー」タブ → 「新規」→「ログオン時」を選択
4. 「操作」タブ → 「新規」→ プログラムに `python`、引数に `C:\kabu_relay\kabu_board_relay.py` を入力
5. 「OK」で保存

## トラブルシューティング

| 症状 | 原因 | 対処法 |
|------|------|--------|
| `APIトークン取得失敗` | kabuステーションが起動していない | kabuステーションを起動してから再実行 |
| `APIトークン取得失敗: 401` | パスワードが間違っている | `KABU_API_PASSWORD` を確認 |
| `送信失敗: 503` | クラウドサーバーが一時停止中 | しばらく待って再試行 |
| 板情報が画面に表示されない | 取引時間外 | 取引時間（9:00〜15:30）に確認 |

## ログファイル

スクリプトと同じフォルダに `kabu_relay.log` が作成されます。問題が発生した場合はこのファイルを確認してください。
