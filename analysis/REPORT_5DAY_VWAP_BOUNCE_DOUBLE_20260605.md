## 悪化の主因分析

### 村田製作所が大幅悪化（0円 → −44,000円）

新しいシグナル（VWAP反発・ダブルボトム）が村田製作所に多数発動し、取引数が0件→5件に増加しましたが、勝率20%（1勝4負）と低調でした。

| 日付 | エントリー理由 | 損益 |
|------|-------------|-----:|
| 6/1 | VWAP反発買い 11:23 | −26,000円（損切り） |
| 6/3 | VWAP反発買い 11:02 | 0円（同値損切り） |
| 6/3 | 既存ロジック 10:05 | +9,600円 |
| 6/4 | VWAP反発買い 11:00 | +2,500円 |
| 6/4 | VWAP反発買い 11:16 | −16,800円（最大保有時間超過） |
| 6/5 | VWAP反発買い 11:23 | −3,700円（同値損切り） |

**問題点**: 11:20強制決済の直前（11:16〜11:28）にエントリーするケースが多発。11:20以降は決済のみ許可のため、エントリー直後に強制決済されて損失になっています。

### SUMCOの6/4に大損（+18,600円 → −27,600円）

Phase37では「空売りトレイリング利確 +18,600円」だったのが、Phase38では「VWAP反発買い → 損切り −27,600円」に変わりました。同じ時間帯に別のシグナルが先に発動してしまい、有利な空売りポジションを取れなかったためです。

## 結論

**VWAP反発・ダブルトップ/ダブルボトムシグナルは5日間では逆効果（−69,900円悪化）でした。**

主な問題点：
1. **11:20強制決済との相性が悪い** — 11:00〜11:20のエントリーが増え、決済時間が短すぎて損失になる
2. **村田製作所との相性が悪い** — 新シグナルが村田に集中し、勝率20%と低調
3. **既存の有利なポジションを上書き** — 先に別シグナルが発動して、より有利なエントリーを逃す

**推奨**: Phase38の変更は採用せず、Phase37（VWAP+ローソク足）の状態に戻すことを推奨します。
